# DAW — Ejercicios

Ejercicios prácticos siguiendo el stack real de Desarrollo de Aplicaciones Web,
orientados al mundo laboral: HTML → CSS → JavaScript → API REST → SQL → Node/Express → React.

Cada ejercicio vive en su propia carpeta, con todo lo que necesita (HTML, CSS, JS) junto.
